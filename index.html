About the game

This game demonstrates an AI algorithm called "A*" (pronounced A star) which enables the dinghy to efficiently collect the yellow and red flags.

A* is an algorithm that helps you find the most efficient route between point A and point B.

You can think of it as a very basic version of Google Maps directions.
